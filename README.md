# Virtual Science Lab

**Phòng Thí Nghiệm Khoa Học Tự Nhiên Ảo** — ứng dụng mô phỏng thí nghiệm Khoa học tự nhiên (Vật lí · Hóa học · Sinh học · Thiên văn) dành cho học sinh THCS.

> Đây là kho phát hành chính thức. Tải bản cài đặt mới nhất tại mục **Releases** bên phải.
> Ứng dụng sẽ **tự động kiểm tra và cập nhật** lên phiên bản mới nhất.

## Tải về

Vào tab **[Releases](../../releases)** → tải file `VirtualScienceLab_x.x.x_x64-setup.exe` → chạy để cài đặt.

## Tính năng

- 55+ module thí nghiệm tương tác trực quan
- Mô phỏng vật lí, phản ứng hóa học, sinh học và thiên văn
- Giao diện tiếng Việt, thân thiện với học sinh THCS
- Tự động cập nhật phiên bản mới
